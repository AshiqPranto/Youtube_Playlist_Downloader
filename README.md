
# YouTube Playlist Downloader

## Overview

YouTube Playlist Downloader is a simple Python script that enables users to download videos from YouTube playlists. I don't find any video downloader by which I can download a playlist easily. Yeah, IDM is the one but it is not free. This script stands out by supporting playlist downloads and providing an easy-to-use command-line interface.

## Features

- Download videos from a specified YouTube playlist.
- Choose a range of video numbers to download within the playlist.
- Specify the directory path to save downloaded videos.

## Getting Started

Follow these steps to use the YouTube Playlist Downloader:

1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/YouTube_Playlist_Downloader.git
   ```

2. Navigate to the project directory:
   ```sh
   cd YouTube_Playlist_Downloader
   ```

3. Run the script:
   ```sh
   python main.py
   ```
